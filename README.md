### 중요사항
1. 이전 여러 번 학습하였으나, 새로운 마음가짐으로 다시 학습 및 코드 주석 추가

  - 주석을 통해 다른 곳에서도 내 Github 접속함을 통해 눈으로 복습 가능
2. 책에 있는 코드를 단순히 따라하는 것이 아니라 나만의 코드로 작성

5. 커널은 jupyter notebook의 파이썬 3.9.12 활용
